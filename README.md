# ScoutFlow

ScoutFlow is an autonomous research-to-action intelligence agent project.

## Setup

```bash
cd scoutflow
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## Run

```bash
python run.py
```

The API runs at:

```text
http://127.0.0.1:8000
```

Health check:

```text
http://127.0.0.1:8000/health
```
